# Phase 1: Programming Fundamentals (Must Be Comfortable)

Before DSA, make sure you can write code quickly.

* Variables and data types
* Loops
* Functions
* Arrays
* Strings
* Recursion
* Time Complexity (Big O)
* Space Complexity

**Understand deeply**

* Best/Average/Worst case
* Amortized complexity
* Recursion tree
* Call stack

---

# Phase 2: Arrays (Highest Priority)

Master every common pattern.

Topics:

* Traversal
* Prefix Sum
* Suffix Sum
* Difference Array
* Sliding Window
* Two Pointers
* Kadane's Algorithm
* Dutch National Flag
* Rotate Array
* Merge Intervals
* Matrix traversal
* Spiral Matrix
* Binary Search on Arrays
* Searching in Rotated Arrays

Problems should include:

* Maximum Subarray
* Product Except Self
* Trapping Rain Water
* Container With Most Water
* Next Permutation

---

# Phase 3: Strings

Topics:

* Character Frequency
* Palindrome
* Two Pointers
* Sliding Window
* HashMap
* String Matching Basics

Algorithms:

* KMP (optional for medium companies)
* Rabin-Karp (basic idea)
* Z Algorithm (optional)

Must solve:

* Longest Substring Without Repeating Characters
* Minimum Window Substring
* Group Anagrams
* Valid Anagram
* Longest Palindrome

# Phase 4: Hashing

Understand when hashing reduces O(n²) to O(n).

Topics:

* HashMap
* HashSet
* Frequency counting
* Prefix Sum + HashMap
* Ordered Map

Must know:

* Two Sum
* Longest Consecutive Sequence
* Subarray Sum Equals K

# How Many Problems?

Quality matters more than raw numbers, but this is a reasonable target:

* Arrays: 50–70
* Strings: 30–40
* Hashing: 20–30
* Linked Lists: 25–30
* Stack & Queue: 30–40
* Binary Search: 30–40
* Trees & BST: 60–80
* Heaps: 20–25
* Recursion & Backtracking: 25–35
* Greedy: 25–30
* Graphs: 50–70
* Dynamic Programming: 60–80
* Trie & Bit Manipulation: 15–20 each
